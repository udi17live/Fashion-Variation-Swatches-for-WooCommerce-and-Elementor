== Description ==

**Fashion Variation Swatches for WooCommerce and Elementor** transforms the default WooCommerce variation dropdowns into beautiful, user-friendly swatches for size and color attributes. Perfect for fashion, apparel stores and any product catalog that needs intuitive variation selection.

== Installation ==

### Automatic Installation

1. Log in to your WordPress admin panel
2. Navigate to **Plugins > Add New**
3. Search for "Fashion Variation Swatches for WooCommerce and Elementor"
4. Click **Install Now** and then **Activate**

### Manual Installation

1. Download the plugin zip file
2. Navigate to **Plugins > Add New > Upload Plugin**
3. Choose the downloaded zip file and click **Install Now**
4. Activate the plugin

### After Installation

1. Go to **WooCommerce > Variation Swatches** to configure settings
2. Navigate to **Products > Attributes** to set up your size and color attributes
3. For color attributes, edit each term and set its color using the color picker
4. Create variable products using these attributes

== Frequently Asked Questions ==

= Does this work with any theme? =

Yes! The plugin is designed to work with any properly coded WordPress theme. It uses standard WooCommerce hooks and includes responsive CSS.

= Can I customize the swatch appearance? =

Absolutely! The plugin includes multiple built-in styles (square, rounded, circle) and extensive CSS classes for custom styling. Elementor widgets include full styling controls.

= Does it work with Elementor? =

Yes! The plugin includes two custom Elementor widgets for size and color swatches with full styling options.

= Will it slow down my site? =

No. The plugin is optimized for performance with minimal CSS/JS and efficient database queries.

= Can I use custom attributes? =

Yes! You can configure which attributes to use for sizes and colors in the plugin settings.

= Does it support translations? =

Yes, the plugin is fully translation-ready with proper internationalization.

= Is it mobile-friendly? =

Completely! The swatches are fully responsive and optimized for touch devices.

= Can customers filter products by size/color? =

Yes! Enable the shop filters option to automatically add size and color filter widgets to your shop page.
